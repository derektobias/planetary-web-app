# Planetary Physics Visualization Web App

Interactive web application for exploring physics and planetary science through real-time simulations and data visualization.

## Tools

### Planetary Comparison Lab
Compare properties of up to two planetary bodies across categories such as physical, orbital, and environmental/surface characteristics. Includes visual representations and size comparisons.

<p align="center">
  <img src="images/planet_comparison_screenshot.JPG" width="750"/>
</p>

### Scenario Comparison Tool
Simulate real-world scenarios (weight, jump height, throw distance, running speed) across different planetary environments with animated visual comparisons and unit switching.

<p align="center">
  <img src="images/scenario_comparison_screenshot.JPG" width="750"/>
</p>

### Conversion Calculator
Perform unit conversions across multiple categories including length, area, volume, mass, time, speed, energy, power, and temperature.

  <img src="images/conversion_calculator_screenshot.JPG" width="750"/>
</p>
